# Changelog

Все значимые изменения проекта фиксируются в этом файле.

Формат версии: **YYYY.MM.DD.NN**

---

## [2026.03.28.01] — 2026-03-28

### Добавлено
- Новый источник **Яндекс Архивы** (`ya.ru/archive/catalog/{каталог}/{N}`)
  - Публичный доступ, авторизация не требуется
  - Получение ID изображения через fetch текущего URL и поиск `"thumb":{"path":"..."}` в ответе
  - Скачивание оригинала через `ya.ru/archive/api/image?id=...&type=original`
  - Имя файла: `fNNNN.jpeg` (номер страницы, дополненный нулями до 4 знаков с префиксом `f`)
- `host_permissions` для `ya.ru` в `manifest.json`

### Изменено
- Архитектура расширения: добавлена поддержка `needsPageScan` — опциональный шаг `scanPage(url)` перед парсингом для источников, требующих получения данных со страницы
- `parseUrl` и `processCurrentTab` принимают параметр `extra` с данными, полученными при сканировании
- `processCurrentTab` стал асинхронным (`async`)
- Убрано дублирование `detectSource` в обработчике `DOMContentLoaded`
- Исправлена неиспользуемая переменная `response` в `downloadImage`

## [2026.03.21.02] — 2026-03-21

### Добавлено
- Источник **FamilySearch** (`familysearch.org`)
  - Аутентификация через cookies браузера
  - Парсинг кода между `:` и `?` в URL
  - Загрузка через `sg30p0.familysearch.org`
- Источник **RusNEB** (`viewer.rusneb.ru`)
  - Публичный доступ
  - Парсинг кода документа и номера страницы
  - Имя файла: `КОД_NNNN.jpeg`
- Service Worker (`background.js`) для загрузки файлов через `chrome.downloads`
- 6 цветовых тем с сохранением выбора в `localStorage`
- Модульная архитектура источников (`sources` + `detectSource` + `parseUrl`)
