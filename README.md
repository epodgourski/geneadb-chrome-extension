## 🎯 О приложении

**Загрузчик документов** — это расширение для Google Chrome, которое помогает быстро скачивать архивные документы из различных источников без необходимости копировать ссылки вручную.

### ✨ Главные возможности

- 🔍 **Автоматический парсинг URL** - расширение самостоятельно определяет источник и извлекает нужные данные
- 📥 **Загрузка документов** - скачивание файлов с использованием текущей сессии браузера
- 🎨 **6 цветовых тем** - красивые и спокойные палитры для работы с архивами
- 🏗️ **Модульная архитектура** - легко добавлять новые источники без изменения основного кода
- 🔐 **Безопасность** - использование cookies браузера вместо хранения пароля

## 🚀 Быстрый старт

### Установка

1. **Скачайте файлы расширения:**
   - `manifest.json`
   - `popup.html`
   - `popup.js`
   - `background.js`
   - `icon128.svg`

2. **Создайте папку** на компьютере (например `geneadb-downloader`)

3. **Поместите файлы** в папку

4. **Откройте chrome://extensions/** в браузере

5. **Включите режим разработчика** (тумблер справа вверху)

6. **Нажмите "Загрузить распакованное расширение"**

7. **Выберите вашу папку** - готово! 🎉

### Использование

1. **Откройте URL** любого поддерживаемого источника в браузере
2. **Кликните иконку расширения** (в правом верхнем углу Chrome)
3. **Расширение автоматически распарсит** информацию
4. **Нажмите "Скачать документ"** или "Скопировать URL"

## 🏛️ Поддерживаемые источники

### 1. FamilySearch
**URL-паттерн:** `familysearch.org/...`
- Парсит подстроку между `:` и `?`
- Использует cookies браузера
- Загружает из API FamilySearch

**Пример:**
```
Вход:  https://familysearch.org/api/data:12345?param=value
Выход: https://sg30p0.familysearch.org/.../3:1:12345/$dist
```

### 2. RusNEB
**URL-паттерн:** `viewer.rusneb.ru/...`
- Парсит код документа и номер страницы
- Публичный доступ (без аутентификации)
- Загружает страницы документов

**Пример:**
```
Вход:  https://viewer.rusneb.ru/ru/000199_000009_003599547?page=1
Выход: https://viewer.rusneb.ru/api/v1/document/000199_000009_003599547/page/1
Файл:  000199_000009_003599547_0001.jpeg
```

### 3. Яндекс Архивы
**URL-паттерн:** `ya.ru/archive/catalog/{каталог}/{N}`
- Публичный доступ (без аутентификации)
- Делает fetch текущего URL и ищет в ответе `"thumb":{"path":"..."}` для получения ID изображения
- Скачивает оригинал через `ya.ru/archive/api/image?id=...&type=original`

**Пример:**
```
Вход:  https://ya.ru/archive/catalog/758796a8-318b-4755-8741-5a37088f7e64/111
Выход: https://ya.ru/archive/api/image?id={imageId}&type=original
Файл:  f0111.jpeg
```

## 🔧 Для разработчиков

### Добавление нового источника

Добавить новый источник **очень просто**! Отредактируйте `popup.js` и добавьте объект конфигурации:

```javascript
const sources = {
  familysearch: { /* ... */ },
  rusneb: { /* ... */ },
  
  mysource: {
    name: 'My Source',
    needsAuth: false,
    detect: (url) => url.includes('mysource.com'),
    parse: (url) => { /* ваша логика */ },
    generateUrl: (parsed) => { /* ваша логика */ },
    getFilename: (parsed) => { /* ваша логика */ },
    displayText: (parsed) => { /* ваша логика */ }
  }
};
```

## 🔒 Безопасность

### Что расширение НЕ делает:
- ❌ Не сохраняет пароли
- ❌ Не отправляет данные на внешние серверы
- ❌ Не просматривает историю браузера
- ❌ Не собирает личные данные

### Как работает аутентификация:
- ✅ Использует `credentials: 'include'` в fetch запросах
- ✅ Переиспользует существующую сессию браузера (cookies)
- ✅ Работает ровно так же, как если бы вы открыли URL в новой вкладке
