# 🎨 Система тем "Загрузчика документов"

## Обзор

Расширение теперь имеет полную систему тем, оптимизированную для архивов и генеалогии. **6 красивых и спокойных цветовых схем**, совместимых с Vuetify.

## 🔄 Как переключать темы

1. **Откройте popup расширения** (кликните иконку расширения)
2. **Нажмите кнопку 🎨** в правом верхнем углу заголовка
3. **Выберите нужную тему** из выпадающего меню
4. **Тема применится мгновенно** и сохранится в localStorage

## 🛠️ Как добавить новую тему

### Вариант 1: Через popup.js (простой способ)

Откройте `popup.js` и добавьте новую тему в объект `themeDefinitions`:

```javascript
const themeDefinitions = {
  // ... остальные темы ...
  
  'theme-7': {
    name: '🌙 Ночной режим',
    colors: {
      primary: '#1a1a1a',
      primaryDark: '#000000',
      accent: '#00ff00',
      light: '#333333',
      lightBg: '#222222',
      text: '#e0e0e0',
      border: '#444444'
    }
  }
};
```

### Вариант 2: Через vuetify-themes.js (для Vuetify)

Если используете Vuetify, добавьте тему в `genealogicalThemes.light`:

```javascript
'theme-7': {
  dark: false,
  colors: {
    background: '#FFFFFF',
    surface: '#222222',
    primary: '#1a1a1a',
    // ... остальные цвета ...
  }
}
```

И добавьте имя в `themeNames`:

```javascript
export const themeNames = {
  // ... остальные темы ...
  'theme-7': '🌙 Ночной режим'
};
```

## 📋 Структура цветов

Каждая тема содержит 7 основных цветов:

| Цвет | Назначение | Пример (тема 1) |
|------|-----------|---|
| `primary` | Заголовок, основной текст | `#5D4A3D` |
| `primaryDark` | Затемнённый основной цвет | `#4A3A2D` |
| `accent` | Кнопки, выделение, ссылки | `#8B6F47` |
| `light` | Светлая версия акцента | `#E8DCC8` |
| `lightBg` | Фон блоков, секций | `#F5F2ED` |
| `text` | Основной текст | `#2C2416` |
| `border` | Границы, разделители | `#D4C4B0` |

## 🎨 CSS переменные

Темы используют CSS переменные, которые автоматически применяются при переключении:

```css
:root {
  --theme-primary: #5D4A3D;
  --theme-primaryDark: #4A3A2D;
  --theme-accent: #8B6F47;
  --theme-light: #E8DCC8;
  --theme-lightBg: #F5F2ED;
  --theme-text: #2C2416;
  --theme-border: #D4C4B0;
}
```

Используйте эти переменные в CSS:

```css
.my-element {
  background: var(--theme-lightBg);
  color: var(--theme-text);
  border: 1px solid var(--theme-border);
}
```

## 💾 Сохранение выбора

Выбранная тема автоматически сохраняется в `localStorage` браузера:

```javascript
localStorage.getItem('geneadb-theme') // например: 'theme-1'
localStorage.setItem('geneadb-theme', 'theme-3') // сохранить тему 3
```

## 🚀 Использование с Vuetify

Файл `vuetify-themes.js` содержит готовые конфиги для Vuetify:

```javascript
import { genealogicalThemes, themeNames, getCurrentTheme } from './vuetify-themes.js';

const vuetify = createVuetify({
  theme: {
    themes: genealogicalThemes.light
  }
});

// Переключить тему
vuetify.theme.global.name = 'theme-3';
```

## ❓ FAQ

**Q: Как сделать тему по умолчанию другой?**
A: Измените строку в `popup.js`:
```javascript
function getCurrentTheme() {
  const saved = localStorage.getItem('geneadb-theme');
  return saved || 'theme-3'; // Вместо 'theme-1'
}
```

**Q: Как создать полностью чёрную тему?**
A: Добавьте в `themeDefinitions` с чёрными цветами (#000000, #1a1a1a и т.д.)

**Q: Сохраняется ли выбор темы?**
A: Да! Выбранная тема автоматически сохраняется и применяется при следующем открытии
